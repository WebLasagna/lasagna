### Getting started 

#### Installation

```bash
# clone the repository
$ git clone https://github.com/WebLasagna/lasagna.git

# change the working directory to lasagna
$ cd lasagna

# install the dependencies
$ npm install
```

#### Start in development mode

```bash
npm run serve
```

#### Build for production

```bash
npm run build
```
